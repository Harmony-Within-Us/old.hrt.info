# DIYエストラジオールパッチに関する情報

## パッチの選択と適用

適切なエストラジオールパッチを選択するには、投与量、接着強度、個々の皮膚感度などの考慮事項が含まれます。 パッチは、通常、きれいで乾燥した髪のない皮膚領域に適用され、刺激を避けるためにアプリケーションサイトの回転をお勧めします。
