# アンチアンドロゲンの概要

**フェミニズム化** アンチアンドロゲンはフェミニズム化に貢献します。 抗アンドロゲンはアンドロゲン(男性ホルモン)の効果を阻害する薬です。 トランスジェンダー個人(MTF)に対するホルモン交換療法(HRT)の文脈では、抗アンドロゲンがテストステロンレベルを抑制するためにしばしば使用される。
